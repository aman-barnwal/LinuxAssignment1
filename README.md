## Task Descriptions

### Task 01 – File & Directory Commands
Created a directory and multiple files using basic commands such as `mkdir`, `touch`, and listed contents using `ls`.

### Task 02 – File Permissions
Modified file permissions using `chmod` to assign read/write/execute privileges and verified using `ls -l`.

### Task 03 – grep Command
Used `grep` to search for specific patterns in a text file and `grep -c` to count occurrences.

### Task 04 – Sorting File Content
Sorted contents of a file using the `sort` command and redirected output into a new file.

### Task 05 – Arithmetic Script
Wrote a shell script for arithmetic operations (+, -, *, /) including division-by-zero handling.

### Task 06 – File Searching
Used the `find` command to locate `.txt` files modified in the last seven days.

### Task 07 – Archiving & Extraction
Compressed the project directory using `tar` and extracted it to validate successful archiving.

### Task 08 – Networking
Checked the system’s IP address with `hostname -I` and tested connectivity using `ping`.

### Task 09 – Largest of Three Numbers
Shell script using conditional logic to identify the largest among three input values.

### Task 10 – Sum of Natural Numbers
Shell script calculating the sum of first N natural numbers using the mathematical formula.
